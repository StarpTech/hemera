API

Global Interface

context$

Will be shared across nested actions.

meta$

Will be shared across nested actions. Data will be transfered.

delegate$

Will be shared across nested actions but the called service don't pass it to the next. Data will be transfered.

auth$

Will be used to store sensitive data e.g plugins which provide authentication or authorization mechanism.

plugin$

Returns the current plugin instance.

  • register Function
  • attributes Object
    • name String
  • parentPlugin String
  • options Object

request$

Returns the current request object.

  • id String
  • type "request" or "pubsub"
  • service String
  • method String

trace$

Returns current tracing informations

  • traceId String
  • parentSpanId String
  • spanId String
  • timestamp Number
  • service String
  • method String
  • duration Number

get plugins

Returns the list of all registered plugins.

get router

Returns the bloomrun instance.

get load

Returns the load instance.

config

Returns the config.

list(pattern: Object, options: Object)

Returns the list of registered actions.

get topics

Returns all registered topics.

get errors

Returns all default error objects.

get transport

Returns the current node-nats instance.

ready(handler: Function)

Ready event.

static createError(name: String)

Create a custom error object.

act(pattern: Object, handler: Function)

Call a service method.

add(pattern: Object, handler: Function)

Register a service method.

remove(topic: String, maxMessages: Number)

Unsubcribe a topic from NATS and remove all pattern from index.

removeAll()

Unsubcribe all registered topics from NATS and removes all pattern from index.

ext(type: String, handler: Function)

Register an extension.

use(plugin: Object, opts: Object)

Register a plugin.

decorate(key: String, object: Object, array: deps)

Expose data which can be accessed from all plugins.

hasDecorator(key: String)

Dheck the presence of a decorator

setOption(key: String, value: Any)

Updates the plugin option value.

setConfig(key: String, value: Any)

Updates the config value.

close()

Gracefully shutdown of all plugins, NATS and hemera.

Server method interface

_actMeta

Returns the metadata of the server method.

  • schema Object
  • pattern Object
  • action Function
  • plugin Object

_topic

Returns the current topic name.

_request

Returns the current request object.

  • payload Object
  • error Object

_response

Returns the current response object.

  • payload Object
  • error Object

_pattern

Returns the pattern of the server method.

Client method interface

_actCallback

Returns the callback of the action.

_cleanPattern

Returns the pattern.

_request

Returns the current request object.

  • payload Object
  • error Object

_response

Returns the current response object.

  • payload Object
  • error Object

_pattern

Returns the pattern of the server method.
